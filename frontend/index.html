IA com Groq

Seu assistente inteligente treinado com LLM para perguntas e análise de dados!

O que é?

Um chat inteligente desenvolvido com a API Groq e LLaMA3, capaz de responder perguntas, manter o histórico de conversas e interpretar comandos complexos com IA.

Funcionalidades

  • Chat com histórico de conversa
  • Interface intuitiva com envio automático ao pressionar Enter
  • Suporte para múltiplas linhas com Shift + Enter
  • Backend com Flask + Groq API
  • Preparado para leitura de CSV e geração de PDF por IA

Aplicações

Ideal para automatizar respostas, interpretar planilhas, gerar relatórios e integrar com pipelines de dados.
